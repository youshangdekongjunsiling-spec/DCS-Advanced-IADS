## 快速开始

### 1. 安装脚本

1. 将 `EA18G_EW_Script_improved_by_flyingsampig.lua` 复制到您的 DCS 任务文件中
2. 在任务编辑器中，添加脚本到 "触发器" → "载入时执行"
3. 确保您的任务中有名为 "Growler" 的 EA-18G 单位

### 2. 基本使用

1. **进入游戏后**，按 `F10` 打开通讯菜单
2. **选择载荷配置**：`F10` → `电子战干扰` → `载荷配置`
3. **开启防御模式**：`F10` → `电子战干扰` → `防御区域干扰` → `开启`
4. **观察效果**：当导弹攻击时，会在聊天窗口显示详细的诱爆判定信息

### 干扰模式

#### 1. 防御区域干扰
- **覆盖范围**：360度全方位保护
- **能量消耗**：100点/周期（5秒）
- **成功率上限**：20%
- **适用场景**：多方向威胁、护航任务

#### 2. 防御扇面干扰
- **覆盖范围**：指定方向扇区（前方/左侧/右侧/后方）
- **能量消耗**：100点/周期（5秒）
- **成功率加成**：+30%（最高26%）
- **适用场景**：已知威胁方向、精确防护

#### 3. 攻击区域干扰
- **功能**：压制敌方SAM系统
- **能量消耗**：15点/周期（5秒）
- **作用范围**：50公里内可见的SAM

#### 4. 点目标干扰
- **功能**：精确干扰指定目标
- **能量消耗**：30点/周期（5秒）
- **选择方式**：F10菜单自动扫描80海里内目标

## 技术原理

### 干扰值计算

**基础公式**：
```
总干扰值 = (ALQ-99数量 × 3) + (ALQ-249数量 × 4)
实际干扰值 = 总干扰值 × 距离系数 × 方向权重
```

**距离系数（平方反比定律）**：
```
距离系数 = (参考距离 ÷ 实际距离)²
参考距离 = 5海里 = 9260米
```

**方向权重**：
- 前方：1.0
- 左右：0.8
- 后方：0.6

### 成功率计算

```
基础成功率 = 实际干扰值 ÷ 100
扇面加成 = 基础成功率 × 1.3（仅扇面模式）
最终成功率 = min(加成后成功率, 上限)
```

**成功率上限**：
- 区域防御：20%
- 扇面防御：26%

## 能量管理

### 消耗机制
- **防御模式**：100点/5秒
- **攻击模式**：15点/5秒
- **点目标模式**：30点/5秒
- **过热保护**：能量低于15点时自动关闭

### 充能机制
- **ALQ-99**：15点/5秒/个
- **ALQ-249**：20点/5秒/个
- **无吊舱**：0点/5秒（不充能）

### 能量策略
1. **高负载配置**充能更快，续航更强
2. **合理规划**开启时机，避免过热
3. **优先使用**扇面模式对付单一威胁方向

## Debug信息说明

脚本会实时显示详细的判定信息：

```
导弹#1判定[扇面+前方] | 距离:2.50海里 | 速度:920km/h | 干扰值:44.0 | 成功率:26.00% | ✓ 诱爆成功 (2 o'clock)
```

**信息解读**：
- **导弹#1**：导弹编号
- **[扇面+前方]**：当前干扰模式
- **距离:2.50海里**：导弹到干扰机的3D距离
- **速度:920km/h**：导弹当前速度
- **干扰值:44.0**：计算得出的实际干扰强度
- **成功率:26.00%**：最终诱爆概率
- **✓ 诱爆成功**：判定结果
- **(2 o'clock)**：导弹相对干扰机的方位

## 故障排除

### 常见问题

**Q: 脚本加载后没有F10菜单？**
A: 检查单位名称是否为"Growler"，确保脚本在任务开始时加载

**Q: 导弹没有被诱爆？**
A: 确认已选择载荷配置且开启了防御模式，检查能量是否充足

**Q: 能量消耗过快？**
A: 避免同时开启多种模式，优先使用能耗较低的攻击模式

**Q: 成功率显示异常？**
A: 检查debug信息中的干扰值，确认距离和方向计算是否正确

### 调试模式
脚本内置详细的debug信息，如需关闭可修改脚本开头的 `DEBUG = false`
