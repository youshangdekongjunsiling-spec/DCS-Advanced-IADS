# EA-18G 咆哮者电子战脚本使用说明

## 概述

本脚本为DCS World中的EA-18G咆哮者电子战机提供了先进的电子战能力，包括革命性的长基线干涉定位系统（LBI）、多频段电子干扰和智能导弹对抗系统。脚本基于真实电子战物理原理设计，提供高度逼真的战术电子战体验。

## 系统需求

- **DCS World** 2.7+
- **MIST脚本库** （必须在本脚本加载前预先加载）
- **推荐机型**：EA-18G Growler
- **推荐地图**：任何支持复杂SAM威胁环境的地图

## 核心功能

### 1. 载荷模拟系统

脚本通过现有武器模拟专业电子战设备：

| 模拟设备 | 使用武器 | 数量要求 | 功能描述 |
|---------|---------|---------|----------|
| **LBI长基线定位吊舱** | AIM-9L | 2枚 | 高精度雷达定位系统 |
| **ALQ-99战术干扰吊舱** | Mk-83 | 每1枚=1个吊舱 | 中功率宽频带干扰 |
| **ALQ-249下一代干扰吊舱** | Mk-84 | 每1枚=1个吊舱 | 高功率定向干扰 |

### 2. 长基线干涉定位系统（LBI/ELINT）

#### 2.1 工作原理
基于真实的长基线干涉定位物理原理，通过测量干扰机在目标雷达视野中移动的角度来确定目标位置。这是现代电子战中最先进的被动定位技术。

#### 2.2 精度等级系统
| 视场角度 | 定位精度 | 圆圈颜色 | 描述 |
|---------|---------|---------|------|
| 0° | ±10 km | 红色 | 初始定位 |
| 1° | ±3 km | 橙色 | 角度积累 |
| 3° | ±500 m | 绿色 | 高精度定位 |
| 10°+ | 精确点位 | 无圆圈 | 最终精确定位 |

#### 2.3 操作步骤
1. **启用ESM系统**：确保携带2枚AIM-9L（模拟LBI吊舱）
2. **选择目标**：通过F10菜单选择要定位的雷达目标
3. **开始定位**：开始定位后保持飞行，系统将实时计算视场角度
4. **角度积累**：飞行过程中系统显示已积累的角度和当前精度
5. **精度提升**：随着角度增加，定位精度自动提升并提示
6. **完成定位**：达到10°视场角度时获得精确点位定位

#### 2.4 技术特性
- **物理真实性**：基于真实长基线干涉测量原理
- **误差模拟**：目标显示在圆内随机位置，模拟真实测量误差
- **实时反馈**：显示已走过角度和当前定位精度
- **多目标支持**：可同时定位多个雷达目标
- **视距要求**：需要保持与目标雷达的视距连接

### 3. 电子干扰系统

#### 3.1 干扰类型
- **区域干扰**：对指定区域内所有雷达进行宽频干扰
- **点目标干扰**：对特定雷达单元进行精确定向干扰

#### 3.2 干扰效能计算
基于真实物理公式：
```
干扰效能 = 干扰功率 × 方向权重 × 距离衰减²
```

**方向权重系数：**
- 正前方：100%
- 左右两侧：80%
- 正后方：60%

**距离效应：**
- 5海里内：解锁平方反比限制，干扰效能大幅提升
- 5海里外：正常平方反比衰减

#### 3.3 吊舱参数
| 吊舱型号 | 单吊舱干扰值 | 特点 |
|---------|-------------|------|
| ALQ-99 | 3 | 成熟可靠，适合区域干扰 |
| ALQ-249 | 4 | 新一代高功率，适合点目标 |

### 4. 智能导弹对抗系统

#### 4.1 自动威胁识别
- 实时监控来袭导弹
- 自动计算导弹方位（时钟方位）
- 智能评估威胁等级

#### 4.2 导弹自毁机制
- **自毁阈值**：干扰值达到100时导弹自毁
- **物理原理**：模拟强电磁干扰对导弹制导系统的影响
- **实时反馈**：显示导弹状态和干扰效果

## 技术原理

### 干扰值计算

**基础公式**：
```
总干扰值 = (ALQ-99数量 × 3) + (ALQ-249数量 × 4)
实际干扰值 = 总干扰值 × 距离系数 × 方向权重
```

**距离系数（平方反比定律）**：
```
距离系数 = (参考距离 ÷ 实际距离)²
参考距离 = 5海里 = 9260米
```

**方向权重**：
- 前方：1.0
- 左右：0.8
- 后方：0.6

### 成功率计算

```
基础成功率 = 实际干扰值 ÷ 100
扇面加成 = 基础成功率 × 1.3（仅扇面模式）
最终成功率 = min(加成后成功率, 上限)
```

**成功率上限**：
- 区域防御：20%
- 扇面防御：26%

## Debug信息说明

脚本会实时显示详细的判定信息：

```
导弹#1判定[扇面+前方] | 距离:2.50海里 | 速度:920km/h | 干扰值:44.0 | 成功率:26.00% | ✓ 诱爆成功 (2 o'clock)
```

**信息解读**：
- **导弹#1**：导弹编号
- **[扇面+前方]**：当前干扰模式
- **距离:2.50海里**：导弹到干扰机的3D距离
- **速度:920km/h**：导弹当前速度
- **干扰值:44.0**：计算得出的实际干扰强度
- **成功率:26.00%**：最终诱爆概率
- **✓ 诱爆成功**：判定结果
- **(2 o'clock)**：导弹相对干扰机的方位

## 故障排除

### 常见问题

**Q: ESM定位无反应**
A: 检查是否携带2枚AIM-9L，确认目标雷达在探测范围内且无地形遮挡

**Q: 干扰效果不明显**
A: 确认干扰吊舱数量充足，检查与目标的距离和相对方位

**Q: 导弹不会自毁**
A: 确认干扰值已达到阈值100，检查干扰机与导弹的相对位置

**Q: 脚本加载失败**
A: 确认MIST脚本已提前加载，检查是否有语法错误
